## Run Locally

Clone the project

```bash
 git clone https://github.com/joahangf11/ClojureLab5-RegularExpressions.git
ClojureLab6-Automata
```

```bash
 cd ClojureLab5-RegularExpressions
```

Be sure to have the most recent modifications

```bash
  git pull
```
